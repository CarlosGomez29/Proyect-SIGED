# **App Name**: ESAC Manager

## Core Features:

- User Management: User authentication and role management (admin, instructor, staff, student).
- Student Registration: Student registry with integrated document uploading (ID, birth certificate, diploma, photos).
- Course Catalog: Course catalog management with details (duration, cost, objectives, content).
- Course Promotion Management: Management for course promotions (start/end dates, capacity).
- Enrollment Management: Enrollment process to register students for courses.
- Grade & Certificate Management: Grade input and digital certificate issuing.
- Payments & Balances: Financial transactions linked to student records, tracking debts and paid sums

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to represent trust and security, aligning with the institution's focus.
- Background color: Light Blue-Gray (#E8EAF6), providing a neutral backdrop that ensures readability and reduces visual fatigue.
- Accent color: Vibrant Orange (#FF7043) for interactive elements, drawing attention and highlighting calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for headlines; Body font: 'Inter' (sans-serif) for body.
- Use professional and clear icons to represent different modules and actions.
- A clean, card-based layout will ensure that large amounts of data are readable and usable.
- Use subtle transitions and animations to enhance user experience, such as loading indicators and form feedback.
